# Stock Box

A web application to manage algirthic trading strategies and allocate stock tickers, with a dashboard for perfomance metrics. 

Platform (cloud)  
  - Serverless lambda
  - Python Flask 
  - MySql DB
  - Redis cache
Platform (on prim)
  - Docker on ARM
  - Kubernates
